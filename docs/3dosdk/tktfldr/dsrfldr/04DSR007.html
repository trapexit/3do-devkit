numsubsmessages

Specifies number of subscriber messages.

Synopsis

numsubsmessages <number_of_messages>

Description

Specifies the number of subscriber messages that should be allocated to run a given stream. If you use this command and writestreamheader in your script, the weaver will put this information into the stream header.

Arguments

number_of_messages
0 to 7FFFFFFF (hex)

Caveats

None.
