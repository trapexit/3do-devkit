NewTextSubscriber

Create Text subscriber thread.

Synopsis

int32 NewTextSubscriber (TextContextPtr *pCtx, DSStreamCBPtr streamCBPtr, int32 deltaPriority)

Description

Creates and initializes a SCel subscriber thread for the calling task. Returns a pointer to a context structure, which the client application uses when making calls to the SCel subscriber.

Arguments

pContext
Address where pointer to Text context structure is returned.
streamCBPtr
Pointer to control block for data stream.
deltaPriority
Relative execution priority of new Text subscriber thread.

Return Value

Returns an error code, or zero if there was no error.

Caveat

This function is part of the Subscriber.lib library file that can currently (Toolkit1.5) be found in Attic:Streaming:2p1:DataStream:Subscribers:Subscriber.lib. It will later be integrated into the main Streaming tree.
