DisposeSemaphore

Deletes a semaphore.

Synopsis

DisposeSemaphore (semaphore)

Description

Deletes a semaphore. DisposeSemaphore() is a macro for DeleteItem():

#define DisposeSemaphore (semaphore) DeleteItem(semaphore)
DisposeSemaphore() is not currently used by the DataStreamer.

Arguments

semaphore
Semaphore to be deleted.

Return Value

Value
Error Condition
(A DeleteItem() item-
None
dependant return value)
(Portfolio error code)
DeleteItem() failed
See "Error Messages" in the 3DO Portfolio Reference Manual for a listing of error messages be returned by DeleteItem().
