ReadDoneBlockFile

Returns status of block file read.

Synopsis

bool ReadDoneBlockFile (Item ioreqItem)

Description

Returns the status of the specified block file read. ReadDoneBlockFile() calls CheckIO() directly to determine whether the Portfolio I/O request is complete.

Arguments

ioreqItem
Item of the I/O request.

Return Value

Value
Error Condition
TRUE
The request has completed
FALSE
The request is still in progress

Associated Files

BlockFile.h

See Also

AsynchReadBlockFile, CheckIO
