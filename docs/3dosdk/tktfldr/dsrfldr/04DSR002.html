dataacqdeltapri

Specifies data acquisition thread priority.

Synopsis

dataacqdeltapri <priority>

Description

Specifies data acquisition thread priority.

The task priority of the data acquisition thread is calculated in relation to the priority of the application that uses the data acquisition task. Therefore, delta priority of the data acquisition thread plus the priority of the application should not exceed 200.

Arguments

delta_priority
Integer. If you add the priority of the application that uses the data acquisition thread and the delta priority, the number must not exceed 210. The higher the number, the higher the priority.
