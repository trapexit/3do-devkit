mediablocksize

Specifies block size of the media. 2048 for CD-ROM.

Synopsis

mediablocksize <blocksize>

Description

Mediablocksize specifies the blocksize of the media (source of the streamed output). The blocksize of most CD-ROM drives is 2048 bytes.

Arguments

blocksize
0 to 7FFFFFFF (hex).

Caveats

None.
