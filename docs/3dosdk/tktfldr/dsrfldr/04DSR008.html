preloadinstrument

Specifies an audio subscriber tag for preloading.

Synopsis

preloadinstrument <audio_data_type>

Description

Specifies an SAudioSubscriber tag name for preloading instruments. This is saves time when the instrument is later needed by a sound in the stream file.

If you use this command, you also have to include writestreamheader. The information is then included in the stream header.

Arguments

audio_data_type
One of the following uncompressed or compressed instrument tags:
Uncompressed instrument tags:

  • SA_44K_16B_S - 44 kHz, 16-bit stereo
  • SA_44K_16B_M - 44 kHz, 16-bit mono
  • SA_44K_8B_S - 44 kHz, 8-bit stereo
  • SA_44K_8B_M - 44 kHz, 8-bit mono
  • SA_22K_8B_S - 22 kHz, 8-bit stereo
  • SA_22K_8B_M - 22 kHz, 8-bit mono
Compressed instrument tags:

  • SA_44K_16B_S_SDX2 - 44 kHz, 16-bit stereo, 2 to 1 compression
  • SA_44K_16B_M_SDX2 - 44 kHz, 16-bit mono, 2 to 1 compression
  • SA_22K_16B_S_SDX2 - 22 kHz, 16-bit stereo, 2 to 1 compression
  • SA_22K_16B_M_SDX2 - 22 kHz, 16-bit mono, 2 to 1 compression
  • SA_44K_16B_M_ADPCM4_22kHz- 16-bit mono, 4:1 compression
  • SA_22K_16B_M_ADPCM4_22kHz- 16-bit mono, 4:1 compression

Caveats

None.
