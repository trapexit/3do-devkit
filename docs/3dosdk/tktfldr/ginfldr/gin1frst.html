Preparing Art for 3DO Titles

Introduction

The 3DO hardware offers graphic artists, video production specialists, and programmers a rich toolbox for preparing a visually appealing title. This chapter, describes how the 3DO system deals with graphic elements, and how you can use these elements effectively.

Note that understanding how the 3DO hardware processes graphics helps you achieve high visual quality and maximum display speed. Using the currently available tools, the artist can actually have almost complete control over what's visible on the screen.
