Compression options

You can use the 3DO-specific compression tool MovieCompress, or the non-3DO-specific Apple tool, ConvertToMovie, to compress QuickTime source movies to QuickTime output. Use MovieCompress or ConvertToMovie if you are doing compression only. Use 3-2 Pulldown for removing composite frames, deinterlacing to 20 or 30 fps, editing (cut, copy, paste), and resizing.

For more information on when and how to use these tools, see Video Tools
