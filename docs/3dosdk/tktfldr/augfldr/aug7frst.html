ARM Assembly Language

ARM Assembly Language is the language which armasm, the ARM Assembler, parses and compiles to produce object code in ARM Object Format. Information on armasm command line options are detailed in The ARM Assembler (armasm). This chapter details ARM Assembly Language, but does not give examples of its use. For such examples refer to the Cookbook.
