Working with the Workbench

To bring up the Workbench and display an art file, follow these steps:

  1. Launch 3DO PostPro by double clicking on the application icon (see Figure 1).

    Figure 1: 3DO PostPro program icon.

  2. Open the document file.

    See Opening and Editing Files for information on how to open a file in 3DO PostPro.

  3. From the Document Window of an open file, drag the Document Proxy onto the Workbench. Figure 2 shows the Document Proxy as it is dragged onto the Workbench.

Figure 2: Document Proxy dragged onto Workbench to display the file.

Note: See Opening and Editing Files for detailed instructions on opening files and editing their characteristics using the Document Window editors.

The file is displayed on the Workbench (see Figure 3) and on the television monitor.

Figure 3: File displayed on Workbench.

If you're working with a cel file, you can reposition it by selecting it and dragging. The location of a cel on the Workbench is saved.
