Converting to a single image

This section shows you how to convert a single Photoshop 2.5 file to a 3DO image.

Note: Photoshop 2.0 file format is also supported.

Follow these steps:

  1. Launch Photoshop and open the desired file.

  2. Scale or resize to 320 x 240 pixels. Larger image sizes cause problems on the NTSC monitor.

    Note: PAL is not currently supported.

  3. If necessary, change the file to RGB color.

    Note: Photoshop grayscale images are not currently supported.

  4. Save the file in Photoshop 2.5 format.

  5. Launch 3DO PostPro.

  6. From the File menu, select Open and open the desired file. A 3DO PostPro Document window appears.

  7. From the Document menu, select Convert to, then 3DO image from the submenu (see Figure 1).

    Figure 1: Convert to 3DO image submenu.

    Note: The Convert to 3DO Texture option is not currently supported.

  8. Type the name for the file in the Save As dialog. Using a .img extension is helpful for future reference. The file is saved as a 3DO image file.

  9. Drag the Document Proxy from the Document window onto the Workbench.

    The image is displayed (see Figure 2).

    Figure 2: File converted to 3DO image .
