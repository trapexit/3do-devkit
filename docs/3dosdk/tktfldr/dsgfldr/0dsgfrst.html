3DO DataStreamer Programmer's Guide

This document helps you understand the 3DO DataStreamer

This document is for C programmers who are preparing titles for the 3DO Station.

How this document is organized

Click on any of the topics below for more information:

  • 3DO DataStreamer Overview provides an overview of the DataStreamer library and associated tools.

  • Preparing and Playing a Simple Stream steps you through moving your data from the Macintosh to synchronized playback on the 3DO Station.

  • Inside Streaming: Basics discusses the different tasks participating in a DataStreamer application and how to initialize and use them.

  • Inside Streaming: Control discusses using control chunks to start and stop the stream and allow user-controlled branching.

  • Debugging and Optimization gives a variety of useful information including tracing, using priorities, and minimizing filler.

  • Creating New Subscribers discusses creating subscribers using the ProtoSubscriber example as a model.
