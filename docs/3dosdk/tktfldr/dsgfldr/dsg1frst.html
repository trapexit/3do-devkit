3DO DataStreamer Overview

The 3DO DataStreamer is a set of data preparation tools and a run-time library of C code that allow retrieval and playback of time-based data. Typical applications include:

  • Playing back a software-compressed movie and its sound track.
  • Synchronizing a cel animation to a sound track.
  • Loading application-specific data while a different audio-visual presentation is running.
