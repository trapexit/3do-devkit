Playing a MIDI file on the 3DO Station

Using ARIA to play a MIDI file on the 3DO Station involves two basic steps, explained in detail in this section:

  • Pasting the PIMap file into the MIDI project document.
  • Loading and playing the files.

Pasting the PIMap file into the MIDI project document

To paste the PIMap file into a MIDI project document, follow these steps:

  1. Open ARIA and close any Patch documents.

  2. Open the PIMap file in your favorite editor, select everything, and copy it onto the clipboard.

  3. From the ARIA File menu, select New MIDI Project.

  4. Click on the first line of the bottom pane of the new document, and paste the PIMap file into that area, as shown in the figure below.

Figure 1: MIDI project with PIMap file.

Loading and playing the files

Once your PIMap file is in the MIDI project window, you have to tell ARIA where to find the MIDI file, then load the PIMap file.

To load the files, follow these steps:

  1. To locate the MIDI file, click on the MIDI File button in the top-left corner of the new MIDI project document and select the MIDI file you want to play.

  2. Finally, click the Load PIMap button, which is also in the top row to the right of the MIDI File box.

    Note: Loading the PIMap and MIDI file takes a while. Progress is reported in the Debugger Terminal window. If you bring the Debugger Terminal window to the foreground while loading the PIMap, the procedure runs more quickly.

  3. When the PIMap finishes loading, click on the Play button in the MIDI project box to load the MIDI file.

  4. Click the Play button (single arrow) in the dialog that appears to play the MIDI file on the 3DO Station.

    Figure 2: MIDI project window with 3DO MIDI Player dialog box.

  5. If you have problems, check the volume on your TV monitor and adjust it if necessary.
