PatchDemo Reference

This document lists the patch file for commands PatchDemo in alphabetical order. Click on any command for more information about it.

  • AttachSample-Attaches a sample to an instrument
  • Connect-Connects an output from one instrument to an input or knob of another instrument
  • DelayLine-Creates a delay line
  • LoadInstrument-Loads a DSP instrument file.
  • LoadSample-Loads an AIFF sample file
  • Tweak-Adjust a knob in an instrument
