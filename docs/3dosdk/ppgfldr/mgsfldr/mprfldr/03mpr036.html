mixer4x2.dsp

Four-channel stereo mixer.

Function

This is a four-channel stereo mixer. The total gain on a side should not exceed $7FFF. Its output is added to the output of the other mixers, then sent to the DAC by head.dsp.

LeftGain - 0 -> $7FFF

LeftGain2 - 0 -> $7FFF

LeftGain3 - 0 -> $7FFF

RightGain0 - 0 -> $7FFF

RightGain1 - 0 -> $7FFF

RightGain2 - 0 -> $7FFF

RightGain3 - 0 -> $7FFF
