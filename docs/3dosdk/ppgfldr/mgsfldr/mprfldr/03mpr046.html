rednoise.dsp

Red noise generator.

Function

This instrument interpolates straight line segments between pseudo-random numbers to produce "red" noise. It is a grittier alternative to the filterednoise.dsp. It is also useful as a slowly changing random control generator for natural sounds.

Knobs

Frequency - 0 -=> $FFFF, default $0300. Controls how often new random numbers are selected. Units are fixed point phase increment. $8000 implies that a new random number will be chosen every other sample frame.

Amplitude - 0 => $7FFF
