mixer8x2.dsp

Eight-channel stereo mixer.

Function

This is an eight-channel stereo mixer. The total gain on a side should not exceed $7FFF. Its output is added to the output of the other mixers, then sent to the DAC by head.dsp.

Knobs

LeftGain0 - 0 -> $7FFF

LeftGain1 - 0 -> $7FFF

...and so on up to...

LeftGain6 - 0 -> $7FFF

LeftGain7" - 0 -> $7FFF

RightGain0 - 0 -> $7FFF

RightGain1 - 0 -> $7FFF

...and so on up to...

RightGain6 - 0 -> $7FFF

RightGain7 - 0 -> $7FFF

Input1

...and so on up to...

Input7
