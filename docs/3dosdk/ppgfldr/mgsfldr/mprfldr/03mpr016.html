directin.dsp

Get audio input.

Function

Outputs the stereo Audio Input signal in Anvil hardware.

Each task that intends to use this instrument must successfully enable audio input with EnableAudioInput() before the instrument can be loaded.

Any number of instances of directin.dsp can be used without interaction.

Outputs

LeftOutput

RightOutput
