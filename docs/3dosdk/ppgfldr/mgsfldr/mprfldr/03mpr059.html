submixer8x2.dsp

Eight-channel stereo submixer.

Function

This is an eight-channel stereo mixer. The total gain on a side should not exceed $7FFF. Output is available directly and can be used as an input to other instruments. It could be used to mix signals before being passed to an effects processor.
