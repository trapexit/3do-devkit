triangle.dsp

Triangle wave generator.

Function

This instrument is a simple triangle wave generator. The waves it produces looks like: /\ /\ /\.

Knobs

Frequency - If you call TweakRawKnob(), the units are phase increments. If you call TweakKnob(), the units are 16.16 fractional hertz.

FrequencyInhertz = RawFreq / (SampleRate * 2)

Amplitude - 0 => $7FFF
