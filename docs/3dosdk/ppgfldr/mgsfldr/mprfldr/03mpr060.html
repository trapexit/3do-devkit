subtract.dsp

Returns the difference between 2 signals.

Function

Outputs the difference between InputA and InputB (A-B). The result is clipped into range. The inputs are signed as usual.

Knobs

InputA

InputB

Outputs

Output
Clipped result of InputA - InputB.
