tj_canon

Uses the juggler to create and play a semi-random canon.

Synopsis

tj_canon <PIMap>

Description

This program shows how to create and play a simple canon using the juggler and score-playing routines.

It creates a single melodic line, and stores it as a sequence to be repeated from 1 to NUMLINES times.

Arguments

PIMap
Name of a PIMap to use. This PIMap should have an instrument defined for program number 1.

Associated Files
