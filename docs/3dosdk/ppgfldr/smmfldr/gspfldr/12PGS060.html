ta_sweeps

Demonstrates tweaking knobs.

Synopsis

ta_sweeps

Description

This program quickly modulates the amplitude and frequency of a sawtooth instrument via tweaking the control knobs repeatedly.

Sweep the frequency smoothly as fast as possible to test Knob speed.

Caveats

This specific sound effect could have been performed entirely within the DSP, freeing up the main processor for other purposes. See the documentation for sawenvsvfenv.dsp.
