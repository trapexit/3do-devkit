ta_tweakknobs

Tweaks the knobs available on an instrument.

Synopsis

ta_tweakknobs <dsp instrument>

Description

Finds the names of all knobs on an instrument and tweaks them to NUMSTEPS * (number of knobs) possible permutations. The program demonstrates how to find and tweak knobs on an instrument, and is useful for testing instrument templates.

Similar functionality is implemented in the Aria tool.

Arguments

dsp instrument
Name of an DSP instrument to tweak. (e.g. sawtooth.dsp).
