Sleep

Sleeps for a number of seconds.

Synopsis

Err Sleep (Item ioreq, uint32 seconds)

Description

Puts the calling task in a wait state for the specified number of seconds. The function does not return until the time has passed.

Arguments

ioreq
IOReq item obtained from GetTimerIOReq(), or zero to have an IOReq dynamically created/deleted during the call.
seconds
Number of seconds to sleep.

Return Value

Zero for success, or a negative error code.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, timerutils.h
