DeleteBasicDisplay

Deletes a display created with CreateBasicDisplay().

Synopsis

Err DeleteBasicDisplay (ScreenContext *sc)

Description

Releases all resources acquired internally by CreateBasicDisplay(), and closes the graphics folio.

Arguments

sc
Pointer to the ScreenContext structure which CreateBasicDisplay() filled in. If this pointer is NULL, the function returns an error code.

Implementation

Library call implemented in lib3do.lib.

Associated Files

lib3do.lib, displayutils.h
