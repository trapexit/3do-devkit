CloseGraphicsFolio

Closes the Graphics folio.

Synopsis

Err CloseGraphicsFolio (void)

Description

This call terminates access to the Graphics folio, freeing any resources allocated by calling OpenGraphicsFolio().

Once the Graphics folio is closed, the task should no longer call any calls from the folio.

Return Value

The call returns 0 if successful or an error code (a negative value) if an error occurs.

Implementation

Convenience call implemented in graphics.lib V21.

Associated Files

graphics.h, graphics.lib
