IsListEmpty

Checks whether a list is empty.

Synopsis

bool IsListEmpty( List *l )

Description

This macro checks whether a list is empty.

Arguments

l
A pointer to the list to check.

Return Value

The macro returns TRUE if the list is empty or FALSE if it isn't.
